# CSS & Flexbox Mastery Project

## Overview
This project is focused on mastering **CSS fundamentals** and building a strong understanding of **Flexbox layout system**.  
The goal is to learn how modern websites are structured and how elements are positioned using flexible layouts.

---

## Learning Goals

By the end of this project, I aim to:

- Understand CSS selectors (class, ID, element selectors)
- Master the difference between classes and IDs
- Learn how CSS specificity works
- Build strong fundamentals in layout design
- Understand Flexbox deeply
- Be able to build responsive layouts without relying on frameworks

---

## Topics Covered

### 1. HTML Structure
- Using semantic HTML elements
- Proper use of `class` and `id`

### 2. CSS Basics
- Selectors
- Colors, fonts, spacing
- Box model (margin, padding, border)

### 3. Flexbox
- `display: flex`
- `justify-content`
- `align-items`
- `flex-direction`
- `gap`
- Flex items behavior
